## ðŸŽ¨ Image Asset Specifications

### Social Media Images
**OG Image (og-image.jpg)**
- Size: 1200 x 630 px
- Format: JPG or PNG
- Max file size: < 1MB
- Content: Logo + tagline "The 1% Standard"
- Colors: Black (#000000) + Yellow (#FFE600)

**Twitter Card (twitter-card.jpg)**
- Size: 1200 x 600 px
- Format: JPG or PNG
- Max file size: < 1MB
- Content: Similar to OG image
- Colors: Black (#000000) + Yellow (#FFE600)

### Favicons
**favicon.ico**
- Size: 32x32 or 16x16 px
- Format: ICO
- Simple "M4" logo

**favicon-16x16.png**
- Size: 16 x 16 px
- Format: PNG
- Transparent background

**favicon-32x32.png**
- Size: 32 x 32 px
- Format: PNG
- Transparent background

**apple-touch-icon.png**
- Size: 180 x 180 px
- Format: PNG
- No transparency (use black background)

**android-chrome-192x192.png**
- Size: 192 x 192 px
- Format: PNG
- Transparent or black background

**android-chrome-512x512.png**
- Size: 512 x 512 px
- Format: PNG
- Transparent or black background

**logo.png**
- Size: 512 x 512 px (recommended)
- Format: PNG
- Transparent background
- For structured data
